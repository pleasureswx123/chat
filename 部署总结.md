## 🚀 Windows 服务器部署步骤

### 第一步:登录 Windows 服务器

使用远程桌面连接到你的 Windows 服务器。

### 第二步:安装必要软件

#### 1. 安装 Git
```
访问: https://git-scm.com/download/win
下载并安装,使用默认选项
```

#### 2. 安装 Node.js
```
访问: https://nodejs.org/
下载 LTS 版本(推荐 18.x 或更高)
安装时确保勾选 "Add to PATH"
```

#### 3. 验证安装
打开 PowerShell,执行:
```powershell
git --version
node -v
npm -v
```

### 第三步:克隆代码

```powershell
# 进入工作目录(例如 C:\)
cd C:\

# 克隆仓库(SSH 方式)
git clone git@github.com:pleasureswx123/chat.git

# 如果 SSH 未配置,使用 HTTPS 方式:
git clone https://github.com/pleasureswx123/chat.git

# 进入项目目录
cd chat\coze_oauth_nodejs_jwt
```

### 第四步:配置应用

```powershell
# 1. 复制配置文件模板
Copy-Item coze_oauth_config.json.example coze_oauth_config.json

# 2. 编辑配置文件
notepad coze_oauth_config.json
```

**填入你的实际配置信息:**
```json
{
  "client_type": "jwt",
  "client_id": "你的客户端ID",
  "public_key_id": "你的公钥ID",
  "private_key": "-----BEGIN PRIVATE KEY-----\n你的私钥内容\n-----END PRIVATE KEY-----",
  "coze_www_base": "https://www.coze.cn",
  "coze_api_base": "https://api.coze.cn"
}
```

### 第五步:一键部署

```powershell
# 运行部署脚本
.\deploy.ps1
```

脚本会自动完成:
- ✅ 检查 Node.js 环境
- ✅ 安装 PM2 进程管理器
- ✅ 检查配置文件
- ✅ 创建日志目录
- ✅ 安装项目依赖
- ✅ 启动服务

### 第六步:验证部署

```powershell
# 查看服务状态
pm2 status

# 查看日志
pm2 logs coze-oauth-service

# 运行健康检查
.\health-check.ps1
```

### 第七步:访问服务

打开浏览器访问: `http://localhost:8080`

## 🌐 配置外网访问(如需要)

### 1. 开放防火墙端口

```powershell
# 以管理员身份运行
New-NetFirewallRule -DisplayName "Coze OAuth Service" `
  -Direction Inbound `
  -LocalPort 8080 `
  -Protocol TCP `
  -Action Allow
```

### 2. 如果使用云服务器

还需要在云服务商控制台配置安全组,开放 8080 端口。
